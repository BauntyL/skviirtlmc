Роль: Ты — опытный Fullstack веб-разработчик, специализирующийся на создании игровых порталов.

Задача: Создать современный, быстрый и стильный многостраничный сайт для ванильного Minecraft сервера Skviirtl Server . Сайт должен быть адаптивным, с темной темой и элементами дизайна в стиле Minecraft (но без излишеств, современный UI).

Информация о сервере:

- Название: Skviirtl Server
- Версия: 1.16.5 – 1.21.11 (поддержка широкого диапазона версий).
- IP: skviirtl.vanilla.cool (с возможностью копирования по клику).
- Концепция: Ванильное выживание с экономикой и упрощенными механиками. Без лишних модов, без кастомных зачарований. Ламповая атмосфера.
- Вход: С любого лаунчера (TLauncher, KLauncher, Mister Launcher и др.), лицензия не обязательна.
Технологический стек:

- Frontend: Next.js (React), Tailwind CSS, Framer Motion.
- Backend: Next.js API Routes (или отдельный Express, если нужно).
- База данных: PostgreSQL / MongoDB (для сайта).
Структура сайта (Страницы):

1. Главная (Home):
   
   - Hero-секция: Логотип/Название, IP, онлайн (фейковый или реальный через API), кнопка "Начать играть".
   - Описание: "Ванильное выживание с экономикой". Упомянуть отсутствие лишнего (кастомных зачарований).
   - Фишки (Features):
     - /rewards — Ежедневные награды.
     - /ah — Аукцион игроков.
     - /seller — Скупщик предметов.
     - /rent — Аренда предметов.
     - /skin — Смена скина прямо в игре.
   - Ссылка на Telegram канал: https://t.me/skviirtl_minecraft (как основное комьюнити).
2. Как начать (Start):
   
   - Простая инструкция:
     1. Скачайте любой лаунчер (TLauncher, KLauncher и т.д.).
     2. Выберите версию от 1.16.5 до 1.21.11.
     3. Добавьте сервер skviirtl.vanilla.cool .
     4. Заходите и регистрируйтесь ( /register пароль пароль ).
3. Кланы (Clans) — Важная страница :
   
   - Описание системы кланов (создание, войны, развитие).
   - Упоминание команды /clan help .
   - Таблица лидеров (Top Clans): Список топ-кланов с колонками:
     - Название клана.
     - Лидер.
     - Со-лидеры.
     - Количество участников / Уровень (KDR).
   - (В будущем здесь будет интеграция с БД плагина SimpleClans или ClanSystem, пока сделай красивую верстку таблицы с мок-данными) .
4. Магазин (Store):
   
   - Покупка привилегий (VIP, Premium и т.д.), кейсов, игровой валюты.
   - Красивые карточки товаров.
5. Личный кабинет (Dashboard) — Требует авторизации на сайте :
   
   - Информация об игроке: Никнейм, Баланс (реальный/игровой), Клан.
   - Смена пароля от сайта .
   - Блок "AiPets Vision" (Задел на будущее): Вкладка или кнопка для запуска веб-интерфейса питомца (демонстрация экрана). Пока может быть неактивной или вести на заглушку.
Функционал:

- Регистрация/Авторизация на сайте (отдельно от игровой, или, в идеале, интеграция с AuthMe , но для начала достаточно отдельной базы сайта).
- Копирование IP по клику.
- Адаптив под мобильные устройства.
